# QuestGenie: Générateur de Flashcards par IA

QuestGenie est une application web qui aide les étudiants à réviser plus intelligemment en générant automatiquement des fiches de révision (flashcards) à partir de leurs notes grâce à l'intelligence artificielle.

## Fonctionnalités

- **Génération de Fiches par IA :** Collez vos notes de cours, et QuestGenie créera des fiches question-réponse pour vous.
- **Synchronisation Cloud :** Grâce à l'authentification Firebase, vos fiches sont sauvegardées sur votre compte et accessibles depuis n'importe quel appareil.
- **Gestion Complète :** Créez, lisez, mettez à jour et supprimez vos fiches manuellement.
- **Mode Révision :** Parcourez vos fiches pour étudier et tester vos connaissances.
- **Développé avec Next.js et Firebase :** Une application web moderne, rapide et évolutive.

## Déploiement

Cette application est configurée pour un déploiement facile avec [Firebase App Hosting](https://firebase.google.com/docs/app-hosting).
