# **App Name**: QuestGenie

## Core Features:

- Material Input Interface: Provide a clean interface for users to input text or notes, which will be processed to generate study materials.
- AI Flashcard Generation Tool: Utilize AI as a tool to automatically generate question-and-answer flashcards from the provided study text or notes.
- Flashcard Editing: Allow students to easily modify and refine the questions and answers generated by the AI to ensure accuracy and relevance.
- Flashcard Persistence: Enable storage and retrieval of flashcard sets, ensuring that student-created content is saved and accessible across sessions.
- Interactive Review Mode: Implement a dynamic review mode where students can practice with their flashcards through an interactive Q&A format.

## Style Guidelines:

- The primary color is a calm and professional blue (#2873BD), promoting focus and clarity.
- The main background color is crisp white (#FFFFFF), enhancing readability and providing a clean aesthetic.
- A subtle, light blue-grey (#F6F7F8) serves as a secondary background color for card elements and distinct sections, maintaining the clean, airy feel.
- A vibrant cyan accent color (#14B8B8) is used for interactive elements and highlights, ensuring clear calls to action and visual interest.
- A single font, 'Inter' (sans-serif), is used for all text to ensure a modern, neutral, and highly readable interface suitable for study materials.
- Use minimalist, clean line icons that convey functionality, such as 'edit', 'add', 'check', and 'delete', supporting a tidy user experience.
- Employ a structured, card-based layout with ample whitespace, organizing content logically and making study material easy to digest.
- Subtle, smooth transition animations for flashcard reveals and navigation to create an engaging yet unobtrusive interactive experience.
